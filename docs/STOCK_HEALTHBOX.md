# Stock Healthbox

O Stock Healthbox resume a condição do ativo antes da análise da opção.

| Indicador | Finalidade |
|---|---|
| ADR | amplitude diária média em percentual |
| ATR | amplitude verdadeira média em preço |
| Variação diária | retorno do ativo no dia |
| Range diário | máxima menos mínima do dia |
| Volume relativo (rVol) | volume atual versus média comparável |
| RSI | força relativa no período configurado |
| RSI 200 | posição/força relativa em horizonte longo definido pela metodologia |
| Volatilidade | dispersão histórica e/ou implícita, identificada separadamente |
| Tendência | alta, baixa, lateral ou indisponível |
| Suporte | nível técnico validado mais próximo |
| Resistência | nível técnico validado mais próximo |
| Distância até suporte | diferença percentual entre preço e suporte |
| Distância até resistência | diferença percentual entre resistência e preço |
| Status geral | síntese baseada somente em indicadores disponíveis |

Cada valor precisa de fonte, fórmula, janela e timestamp. Se faltar dado crítico, o status será **“score não calculado”**.
